Идея проекта

Реализовать Канбан-доску - инструмент для планирования и управления проектом с помощью карточек с задачами.

Ключевые требования

  1. Отображение профиля пользователя
  2. Создание канбан-доски
  3. Создание групп карточек
  4. Создание карточки по названию
  5. Добавление комментариев к карточке
  6. Командная работа над одной доской

Основные сущности

Доска — самая круппная модель, объединяющая группы и задачи

Задача — самая маленькая, но самая основная модель, именно она определяет чем тебе стоит сегодня заняться
